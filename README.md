# X11 HELPERS

Small utilities for the X11 desktop.

## Help

x-color

    Usage: x-color
    
    Select a point in the screen and copy it's color value
    with the #RRGGBB format.

x-screenshot

    Usage: x-screenshot
    
    Select a window and take a screenshot.
